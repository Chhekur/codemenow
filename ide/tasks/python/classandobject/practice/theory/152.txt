Class Syntax<br><br>A basic class consists only of the class keyword, the name of the class, and the class from which the new class inherits in parentheses. (We'll get to inheritance soon.) For now, our classes will inherit from the object class, like so:<br>class NewClass(object):<br>  # Class magic here<br>This gives them the powers and abilities of a Python object. By convention, user-defined Python class names start with a capital letter.
